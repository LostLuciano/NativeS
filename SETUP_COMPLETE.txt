================================================================================
                        TROUBLESHOOTING
================================================================================

PROBLEM: "Model not found" error
SOLUTION:
1. Verify models in MusicStemNative/Models/
2. Check models are in Copy Bundle Resources
3. Clean build: Cmd+Shift+K
4. Rebuild

PROBLEM: Build fails with linker error
SOLUTION:
1. Delete DerivedData:
   rm -rf ~/Library/Developer/Xcode/DerivedData
2. Clean build folder: Cmd+Shift+K
3. Rebuild

PROBLEM: App crashes on separation
SOLUTION:
1. Check available memory
2. Try light model instead
3. Reduce input audio length
4. Check input shape matches model

PROBLEM: Models not loading at runtime
SOLUTION:
1. Verify models in app bundle
2. Check model names in code
3. Verify model files not corrupted
4. Check compute units configuration
