   - Complete API coverage

================================================================================
                              PIPELINE STAGES
================================================================================

Stage 1: Loading (5%)
   - Load audio file metadata
   - Get duration and sample rate
   - Verify file format

Stage 2: Decoding (15%)
   - Extract PCM samples
   - Handle multiple formats
   - Detect channels

Stage 3: Resampling (25%)
   - Convert to 44.1kHz stereo
   - Linear interpolation
   - Preserve quality

Stage 4: Normalization (25%)
   - Find peak amplitude
   - Normalize to 0.95
   - Prevent clipping

Stage 5: STFT (35%)
   - Compute FFT
   - FFT size: 4096/2048
   - Hann window
   - Complex spectrogram

Stage 6: Inference (50%)
   - Stack stereo channels
   - Chunk spectrogram
   - Run CoreML model
   - Select model (standard/light)

Stage 7: iSTFT (70%)
   - Reconstruct time domain
   - Inverse FFT
   - Overlap-add
   - Window compensation

Stage 8: Writing (85%)
   - Write M4A files
   - Create stems directory
   - Preserve quality

Stage 9: Validation (95%)
   - Verify stem files
   - Check file sizes
   - Validate audio data

Stage 10: Analysis (100%)
   - Create analysis.json
   - Store metadata
   - Cache for future use
